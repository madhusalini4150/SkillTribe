# 🚀 SkillTribe
![SkillTribe Homepage](static/images/homepage.png)

SkillTribe is a modern skill-sharing platform where users can learn, teach, and connect with others.

🌐 Live Demo: https://skilltribe.onrender.com/

## ✨ Features

- 👤 User Authentication
- 🔒 Secure Login
- 💬 Real-time Chat
- 💳 Razorpay Payments
- 📤 File Uploads
- 📱 Responsive Design
