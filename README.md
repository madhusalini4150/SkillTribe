# 🚀 SkillTribe

SkillTribe is a modern skill-sharing platform where users can learn, teach, and connect with others.

🌐 Live Demo: https://skilltribe.onrender.com/

## ✨ Features

- 👤 User Authentication
- 🔒 Secure Login
- 💬 Real-time Chat
- 💳 Razorpay Payments
- 📤 File Uploads
- 📱 Responsive Design

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Python
- Flask
- Flask-SocketIO

### Database
- SQLite
- PostgreSQL

### Deployment
- Render
